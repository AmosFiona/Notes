\subsubsection{$\frac{0}{0}$ 型}
	\begin{enumerate}
		\item 无穷小相减$\Rightarrow$泰勒公式
		\item 遇见$\sqrt{\Box}$ 先有理化后观察
		\item 四则运算与洛必达法则配合
		\item 变限积分所表示的函数求极限采用洛必达法则\\
			\quan{1} 积分号里面含$x$拆项到积分号外\\
			\quan{2} $f$里含有$x$，需要用变量代换，将其变化到积分限中才可以用积分求导定理
	\end{enumerate}
